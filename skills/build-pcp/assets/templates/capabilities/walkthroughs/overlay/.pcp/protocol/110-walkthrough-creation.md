# Walkthrough creation

- Build finished-project guidance incrementally from real user questions and verified behavior.
- Rewrite each question into a concise professional FAQ heading without changing its intent.
- Answer with the smallest complete explanation, commands, examples, and cautions needed.
- Verify instructions against the current project before adding them.
- Append the durable Q&A to the owning numbered walkthrough and link it from the relevant index.
- Update or supersede stale answers when behavior changes; do not preserve conversation transcripts as documentation.
