# Build and tooling

## Supported environment

- Record required tool names, compatible versions, and operating-system constraints.
- Never record machine-specific installation paths or credentials.

## Commands

- Document verified install, build, test, lint, format, run, and deployment commands.
- Distinguish fast unit checks from the complete release or milestone gate.

## Configuration

- Name required configuration files and non-secret environment variables.
- Record known build lessons only after they have been reproduced.
